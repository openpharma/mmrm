# Get Prediction Variance

Get prediction variance with given fit, `tmb_data` with the Monte Carlo
sampling method.

## Usage

``` r
h_get_prediction_variance(object, nsim, tmb_data)
```

## Arguments

- object:

  (`mmrm_tmb`)  
  the fitted MMRM.

- nsim:

  (`count`)  
  number of samples.

- tmb_data:

  (`mmrm_tmb_data`)  
  object.

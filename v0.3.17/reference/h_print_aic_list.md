# Printing AIC and other Model Fit Criteria

This is used in
[`print.summary.mmrm()`](https://openpharma.github.io/mmrm/reference/mmrm_methods.md).

## Usage

``` r
h_print_aic_list(aic_list, digits = 1)
```

## Arguments

- aic_list:

  (`list`)  
  list as part of from
  [`summary.mmrm()`](https://openpharma.github.io/mmrm/reference/mmrm_methods.md).

- digits:

  (`number`)  
  number of decimal places used with
  [`round()`](https://rdrr.io/r/base/Round.html).

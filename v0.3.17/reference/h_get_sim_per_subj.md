# Get simulated values by patient.

Get simulated values by patient.

## Usage

``` r
h_get_sim_per_subj(predict_res, nsub, nsim)
```

## Arguments

- predict_res:

  (`list`)  
  from
  [`h_get_prediction()`](https://openpharma.github.io/mmrm/reference/h_get_prediction.md).

- nsub:

  (`count`)  
  number of subjects.

- nsim:

  (`count`)  
  number of values to simulate.

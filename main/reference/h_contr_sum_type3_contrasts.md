# Construct Preliminary Contrast Matrices for Type III Tests Assuming Sum Contrasts

Construct Preliminary Contrast Matrices for Type III Tests Assuming Sum
Contrasts

## Usage

``` r
h_contr_sum_type3_contrasts(object)
```

## Arguments

- object:

  (`mmrm`)  
  the fitted MMRM.

## Value

A `list` of contrast matrices, which are just row-wise term subsets of
an overall identity matrix.

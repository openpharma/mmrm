# Printing MMRM Function Call

This is used in
[`print.summary.mmrm()`](https://openpharma.github.io/mmrm/reference/mmrm_methods.md).

## Usage

``` r
h_print_call(call, n_obs, n_subjects, n_timepoints)
```

## Arguments

- call:

  (`call`)  
  original
  [`mmrm()`](https://openpharma.github.io/mmrm/reference/mmrm.md)
  function call.

- n_obs:

  (`int`)  
  number of observations.

- n_subjects:

  (`int`)  
  number of subjects.

- n_timepoints:

  (`int`)  
  number of timepoints.

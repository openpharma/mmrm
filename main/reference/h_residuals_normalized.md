# Calculate normalized residuals

This is used by
[`residuals.mmrm_tmb()`](https://openpharma.github.io/mmrm/reference/mmrm_tmb_methods.md)
to calculate normalized / scaled residuals.

## Usage

``` r
h_residuals_normalized(object)
```

## Arguments

- object:

  (`mmrm_tmb`)  
  the fitted MMRM.

## Value

Vector of residuals

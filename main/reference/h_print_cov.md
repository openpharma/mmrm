# Printing MMRM Covariance Type

This is used in
[`print.summary.mmrm()`](https://openpharma.github.io/mmrm/reference/mmrm_methods.md).

## Usage

``` r
h_print_cov(cov_type, n_theta, n_groups)
```

## Arguments

- cov_type:

  (`string`)  
  covariance structure abbreviation.

- n_theta:

  (`count`)  
  number of variance parameters.

- n_groups:

  (`count`)  
  number of groups.

# Calculate Pearson Residuals

This is used by
[`residuals.mmrm_tmb()`](https://openpharma.github.io/mmrm/reference/mmrm_tmb_methods.md)
to calculate Pearson residuals.

## Usage

``` r
h_residuals_pearson(object)
```

## Arguments

- object:

  (`mmrm_tmb`)  
  the fitted MMRM.

## Value

Vector of residuals.
